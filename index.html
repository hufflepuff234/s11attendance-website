IT11 S11 Attendance Web System

The web system introduces a standalone attendance system for IT11S11 students. This features a full on CRUD application where students can add themselves inside the data table by filling out several information such as their student number, last name, first name, time they attended the class, as well as their class number. This allows teachers to eliminate the need to pass papers to the students so that they can fill for attendance.
